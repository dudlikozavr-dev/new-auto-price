# PriceMatrix — Система управления прайс-листами поставщиков

## Что это

Веб-приложение для автоматизации работы с прайс-листами поставщиков и синхронизации товарной базы с интернет-магазином на платформе InSales (carstvo-sna.ru).

## Для кого

Владелец интернет-магазина, который работает с несколькими поставщиками и вручную обновляет цены и остатки на сайте.

---

## Функции системы

### 1. Загрузка прайс-листов
- Ручная загрузка файла с компьютера
- Поддерживаемые форматы: CSV, Excel (.xls, .xlsx), YML (Яндекс.Маркет), XML, JSON
- Получение прайса через API поставщика
- Загрузка прайс-листов любой структуры и сложности (гибкий маппинг колонок)

### 2. PriceMatrix — единая база товаров
- Главный прайс как единый справочник всех товаров
- Сводный прайс: наличие и минимальная цена по всем поставщикам
- Устранение дубликатов одинаковых товаров от разных поставщиков
- Список новых товаров, появившихся в последнем прайсе

### 3. Связи товаров
- Автоматическое связывание товаров по артикулу, штрихкоду или названию
- Ручное связывание (сопоставление) товара поставщика с позицией в главном прайсе
- Редактирование и разрыв связей

### 4. Цены и остатки
- Обновление остатков из прайсов поставщиков
- Нахождение лучшей (минимальной) цены среди поставщиков
- Наценка на цену поставщика: фиксированная, процентная, по правилам
- Вся валюта — рубли (конвертация не предусмотрена)

### 5. Выгрузка данных
- Экспорт в форматы: Excel, CSV, XML, YML, JSON
- Формирование собственного прайс-листа магазина, если его ещё нет

### 6. Синхронизация с InSales
- Выгрузка готового прайса (цены + остатки) в InSales через API
- Обновление существующих товаров на сайте carstvo-sna.ru
- Загрузка актуальных данных и контента (описания, фото) прямо на сайт

---

## Чего НЕ будет

- Мультивалютности и конвертации курсов — только рубли
- Поддержки других платформ, кроме InSales
- Автоматического парсинга сайтов поставщиков (только файлы и API)
- Управления заказами, логистикой, складом
- Встроенного каталога поставщиков
- Мобильного приложения
- Многопользовательского доступа с разграничением прав (на первом этапе)

## Этапы реализации

### Этап 1 — Ручная загрузка прайса (MVP)
- [ ] Загрузка файла с компьютера (CSV, Excel, YML, XML)
- [ ] Настройка маппинга колонок (какая колонка — название, артикул, цена, остаток)
- [ ] Сохранение данных в базу (PriceMatrix)
- [ ] Просмотр и фильтрация загруженных товаров
- [ ] Экспорт в Excel/CSV

### Этап 2 — Работа с несколькими поставщиками
- [ ] Несколько поставщиков в системе
- [ ] Связывание товаров (дедупликация)
- [ ] Сводный прайс с минимальной ценой
- [ ] Наценка и формирование цены продажи

### Этап 3 — Выгрузка в InSales
- [ ] Подключение к API InSales
- [ ] Обновление цен и остатков на сайте
- [ ] Загрузка новых товаров

### Этап 4 — Автоматизация
- [ ] Получение прайсов через API поставщиков
- [ ] Расписание автоматического обновления

## Технический стек (предварительно)

- **Frontend:** React / Next.js
- **Backend:** Node.js (Express) или Python (FastAPI)
- **База данных:** PostgreSQL
- **Интеграция:** InSales API
- **Форматы парсинга:** xlsx, csv-parser, xml2js, js-yaml (для YML)
