# Чек-лист проекта PriceMatrix

От простого к сложному. Каждый шаг — законченный результат который можно использовать.

---

## Шаг 1 — Разобраться с данными
- [ ] Открыть `ostatki-Platina.xlsx` и понять структуру: где артикул, цена, остаток, размер
- [ ] Открыть `price-carstvo-sna.xls` и найти колонки: артикул (col 23), штрихкод (col 24), цена (col 26), остаток (col 29)
- [ ] Вручную найти 5 совпадающих артикулов в обоих файлах
- [ ] Убедиться что логика работает: первое слово из `ми4447 l бело-голубой` = `ми4447`

## Шаг 2 — Google Таблица: структура
- [ ] Создать Google Таблицу с тремя листами: `Поставщик`, `Мой прайс`, `Сводная`
- [ ] Вручную скопировать 50 строк из прайса поставщика на лист `Поставщик`
- [ ] Вручную скопировать 50 строк из своего прайса на лист `Мой прайс`
- [ ] Убедиться что колонки совпадают с ожидаемыми

## Шаг 3 — Google Таблица: связка
- [ ] На листе `Сводная` написать формулу VLOOKUP/INDEX-MATCH: найти цену поставщика по базовому артикулу
- [ ] Добавить формулу извлечения базового артикула: `=LEFT(A2, FIND(" ", A2&" ")-1)`
- [ ] Проверить на 50 строках: сколько совпало, сколько нет
- [ ] Добавить второй метод — связка по штрихкоду через IFERROR

## Шаг 4 — Google Таблица: наценка и цена продажи
- [ ] Добавить колонку "Цена поставщика" (из листа Поставщик)
- [ ] Добавить колонку "Наценка %" — одно значение для всех (например 40%)
- [ ] Добавить колонку "Цена продажи" = Цена поставщика × (1 + Наценка%)
- [ ] Добавить колонку "Старая цена" (из моего прайса)
- [ ] Подсветить строки где цена изменилась (условное форматирование)

## Шаг 5 — Google Таблица: фильтры и статусы

## Шаг 7 — Google Таблица: Apps Script — парсинг файла
